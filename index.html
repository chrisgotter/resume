Hi! This is my third design for OSWD, with CSS and XHTML 1.0 Transitional Validation. You can do whatever you want with this template, just keep the Hosting Colombia link at the bottom. Enjoy!

Hi! I am another guy who's programming teacher directed him toward this excelent template. In light of the recent adoption of HTML 5 I decided to endeavor to make this template a little more HTML 5'ish. This edition looks nearly identical to the one which inspired it but has been constructed with HTML 5 a more diverse set of tags. I externalized nearly all display formatting to where it is currently being applied in the css document. Quite possibly my proudest achivement in this exercise has been to make the page scalable. The settings to do so can be found in the body definition in mycss file. Finally, I would like to issue a challenge to the reader: first, I used 2 fieldsets in my head section; these are improperly used but I could not find a more kosher way to get those sections to line up so-try to find the right tags/css attributes to make that happen while keeping the page fluid. Second, the only remaining idea that I could think of to improve the viability of this template is to format the tie logo with css such that the user could upload a photo of the correct dimensions and it would display properly (add the shadow, and slide the background underneath the image). I am primarally interested in entirely css solutions but if you can do so with javascript I would find that interesting as well. Please let me know when you figure it out.
